# MooveServer
Server to display bus time table and API

### Requirements
* Node 7.6+
* Yarn
* Generated a sqlite db from `https://github.com/ChamGeeks/xlsToGTFS`

### Install and run
* `git clone git@github.com:ChamGeeks/MooveServer.git`
* `cd MooveServer`
* Copy the database file `GTFS.sqlite` to root
* `yarn`
* `yarn run dev`
